2. SSTF (Shortest Seek Time First)

Dari 1234, pilih yang jaraknya paling dekat (misal 1300 lebih dekat dibanding 1600)
SSTF selalu memilih track yang paling dekat dari posisi head saat ini. Head mencari permintaan yang memiliki jarak minimum dari posisinya, lalu berpindah ke sana.

Head bergerak ke track terdekat dulu.
Hasil Jarak : 6860 track
66 + 0 + 150 + 150 + 0 + 175 + 125 + 50 + 50 + 950 + 0 + 225 + 125 + 0 + 100 + 100 + 75 + 225 + 60 + 56 + 9 + 20 + 25 = 2636 track

3. SCAN
Head bergerak ke kanan (naik) sampai akhir (track 2000), lalu balik.
Jarak sama dengan SSTF di kasus ini.
Total lintasan SCAN = 2636 track

4. LOOK
Seperti SCAN, tapi berhenti di track tertinggi permintaan (2000) tanpa ke ujung disk kalau tidak perlu.
Jarak sama dengan SCAN.
Total lintasan LOOK = 2636 track

5. C-SCAN (Circular SCAN)
Head jalan naik terus sampai 2000, lalu lompat ke 0, lanjut.
jarak :
66 + 150 + 150 + 0 + 175 + 125 + 50 + 50 + 2000 + 30 + 25 + 20 + 9 + 56 + 60 + 225 + 75 + 100 + 100 + 0 + 125 + 225 + 0 = 3611 track

6. C-LOOK
Seperti C-SCAN, tapi lompat ke track terkecil permintaan, bukan ke 0.
Jarak sama dengan C-SCAN.
Total lintasan C-LOOK = 3611 track
